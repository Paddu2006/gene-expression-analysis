# Gene Expression Analysis

An R-based RNA-seq gene expression analysis tool that identifies
differentially expressed genes and generates publication-quality visualizations.
Built as Phase 2 - Project 2 of my bioinformatics-to-tech portfolio.

## Features
- Simulates realistic RNA-seq count data
- Identifies differentially expressed genes
- Volcano plot visualization
- PCA plot for sample clustering
- Saves results to CSV file

## Results
Upregulated genes   : 17
Downregulated genes : 20
Not significant     : 163
Total genes analyzed: 200

## Visualizations
- volcano_plot.png — shows up and downregulated genes
- pca_plot.png — shows separation between control and treated samples
- results.csv — complete gene expression results

## Tech Stack
- R 4.5.1
- ggplot2
- RColorBrewer
- Git + GitHub

## Usage
Rscript gene_expression.R
